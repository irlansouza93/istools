ISTools Plugin for QGIS
ISTools (Irlan Souza Tools) é um conjunto de ferramentas de vetorização para QGIS, projetado para facilitar operações com vetores.
Ferramentas Incluídas

Extend Lines: Estende linhas soltas até intersectar com outras linhas na mesma camada.
Como usar: Selecione uma camada de linhas, escolha uma ou mais feições e clique no ícone "Extend Lines" na toolbar.


Polygon Generator: Gera polígonos a partir de linhas ou áreas ao redor de um ponto central clicado.
Como usar: Certifique-se de que há camadas de linhas ou polígonos no projeto. Clique no ícone "Polygon Generator", clique no mapa para definir o centro do polígono, ou clique com o botão direito para cancelar.



Requisitos

QGIS 3.0 ou superior
Camadas de linhas ou polígonos (dependendo da ferramenta)

Instalação

Baixe o plugin do repositório oficial do QGIS ou instale via ZIP.
Ative o plugin em Plugins > Manage and Install Plugins.
Os ícones das ferramentas aparecerão na toolbar.
