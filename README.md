## 📋 Sobre

ISTools (Irlan Souza Tools) é um plugin QGIS que oferece ferramentas avançadas para vetorização, simplificando operações comuns e aumentando a produtividade em trabalhos com dados geoespaciais.

## ✨ Funcionalidades

### Extend Lines 📏
Ferramenta inteligente para estender linhas até sua intersecção com outras feições.

**Características:**
- Extensão automática de linhas soltas
- Detecção precisa de intersecções
- Suporte a múltiplas seleções
- Preservação da topologia

### Polygon Generator 🔷
Gera polígonos automaticamente a partir de linhas ou áreas delimitadoras.

**Características:**
- Geração com um clique
- Suporte a múltiplas camadas
- Validação de geometria
- IDs únicos para rastreamento

## 🚀 Instalação

1. Abra o QGIS
2. Navegue até `Plugins > Gerenciar e Instalar Plugins`
3. Pesquise por "ISTools"
4. Clique em "Instalar"

## 🔧 Uso

### Extend Lines
1. Selecione uma camada de linhas
2. Selecione as linhas que deseja estender
3. Clique no ícone Extend Lines na barra de ferramentas
4. As linhas serão automaticamente estendidas

### Polygon Generator
1. Certifique-se de ter camadas de linhas ou polígonos no projeto
2. Clique no ícone Polygon Generator
3. Clique no mapa para definir o centro do polígono
4. O polígono será gerado automaticamente

## ⚙️ Requisitos

- QGIS 3.0 ou superior
- Camadas vetoriais (linhas ou polígonos)
